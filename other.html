GUN SHRIMP

Perlenspiel is a basic game programming engine made in Javascript by Brian Moriarty. With just a grid of beads for a screen and a line of status text, it's designed to be constrictive in a way that encourages creativity. It's used in WPI's Digital Game Design course (IMGD 2900). For the final project, I was required to construct a Metaphor Game, in which the game mechanics are tied to some sort of visual indicator. For this project, I created Gun Shrimp, a simple arcade-style game where you play as a shrimp trying to shoot the enemy crab as much as possible without it shooting you back. The game gets increasingly difficult as you score points, as the crab becomes faster and more aggressive.
